Руководство по эксплуатации программы:

1. Интерфейс программы:
После запуска программы откроется окно набора таблиц данных (Склад, Сотрудники, Товары, Поставки).
Каждая таблица имеет базовый набор кнопок для манипуляции с данной таблицей.
На нижней панели тоже располагаются кнопки, действующие не на конкретную таблицу, а на всю программу.

2. Гайд по всем функциям программы:
  2.1 Добавление данных:
  На каждой вкладке таблицы имеется кнопка "Добавить".
  Нажмите кнопку "Добавить" для открытия окна ввода новых данных.
  Введите необходимую информацию в соответствующие поля.
  Нажмите "Добавить", или на клавишу ENTER чтобы добавить данные в таблицу. Иначе - "Отмена".

  2.2 Удаление данных:
  Для удаления записи выберите одну строку в таблице.
  Нажмите кнопку "Удалить".
  Подтвердите удаление, нажав на "Да" или клавишу ENTER или откажитесь, нажав "Нет".

  2.3 Редактирование данных:
  Для редактирования существующей записи дважды выберите строчку, которую вы хотите изменить.
  Внесите необходимые изменения в открывшееся окно редактирования элемента.
  Примите или отклоните изменения, нажав соотвутсвующую кнопку.

  2.4 Обновление данных:
  Для обновления данных нажмите кнопку "Обновить" под соответствующей таблицей

  2.5 Руководство:
  Вы уже умеете открывать руководство, не так ли?

  2.6 О программе:
  Для просмотра информации о программе нажмите на кнопку "О программе".

  2.7 Выполнение поставки:
  Для того, чтобы выполнить поставку из таблицы "Поставки", нужно нажать на кнопку "Выполнить поставку" и подтвердить выполнение.

  2.8 Выход из программы:
  Для выхода из программы намите кнопку "Выход" в правой стороне и подтвердите действие.
